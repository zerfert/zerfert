### 🖐️ Sistema de Control de Computadoras por Gestos
> **🏆 Proyecto de investigación presentado en el congreso CIECT XX**
>
> Sistema que permite controlar una computadora mediante gestos de la mano, con procesamiento seguro de datos visuales en tiempo real.
>
> **Stack:** `Python` · `OpenCV` · `MediaPipe` · `Visión Computacional`
>
> 🔹 Detección y tracking de gestos en tiempo real  
> 🔹 Procesamiento seguro de datos visuales (privacidad por diseño)  
> 🔹 Arquitectura optimizada para baja latencia  
> 🔹 Presentado y validado en congreso académico  
>
> *📂 Repositorio próximamente — en proceso de preparación para publicación open-source.*

### 🧠 [API de Predicción de Depresión Estudiantil](https://github.com/zerfert/depression-api)
> API REST potenciada con Machine Learning que predice la probabilidad de depresión en estudiantes a partir de factores académicos, personales y de bienestar.
>
> **Stack:** `Python` · `FastAPI` · `Scikit-Learn` · `Docker` · `Azure` · `GitHub Actions`
>
> 🔹 Modelo de Regresión Logística con selección de features RFE (~85% accuracy)  
> 🔹 Documentación interactiva con Swagger UI  
> 🔹 Despliegue con Docker & Azure App Service  
> 🔹 Pipeline CI/CD completo con GitHub Actions
