## Proyectos Destacados

### Sistema de Control de Computadoras por Gestos
> **Proyecto de investigacion presentado en el congreso CIECT XX**
>
> Sistema que permite controlar una computadora mediante gestos de la mano, con procesamiento seguro de datos visuales en tiempo real.
>
> **Stack:** `Python` `OpenCV` `MediaPipe` `Vision Computacional`
>
> - Deteccion y tracking de gestos en tiempo real
> - Procesamiento seguro de datos visuales (privacidad por diseno)
> - Arquitectura optimizada para baja latencia
> - Presentado y validado en congreso academico
>
> *Repositorio proximamente -- en proceso de preparacion para publicacion open-source.*

---

### [API de Prediccion de Depresion Estudiantil](https://github.com/zerfert/depression-api)
> API REST potenciada con Machine Learning que predice la probabilidad de depresion en estudiantes a partir de factores academicos, personales y de bienestar.
>
> **Stack:** `Python` `FastAPI` `Scikit-Learn` `Docker` `Azure` `GitHub Actions`
>
> - Modelo de Regresion Logistica con seleccion de features RFE (~85% accuracy)
> - Documentacion interactiva con Swagger UI
> - Despliegue con Docker y Azure App Service
> - Pipeline CI/CD completo con GitHub Actions
